# Portfolio Website

Personal portfolio website showcasing projects, writing, and community/speaking activity. Built to load fast, read well on mobile, and present work clearly with clean UI and consistent components.

## Live Site
- Website: (https://tobechiduru.vercel.app/)
- Repo: (https://github.com/Tobe01/Portfolio)

## Sections
- Hero + About
- Projects (selected work + details)
- Articles / Writing 
- Speaking & Community
- Contact

## Key Features
- Responsive layout (mobile-first)
- Reusable UI components and consistent styling
- Fast navigation with client-side routing 
- SEO-friendly metadata
- Accessible UI patterns (focus states, readable contrast)

## Tech Stack
- React + TypeScript + Shadcn UI + Cloudinary
- Styling: Tailwind / CSS Modules / Styled Components
- Routing: React Router 
- Deployment: Vercel

## Getting Started

### Prerequisites
- Node.js (LTS)
- npm / yarn / pnpm

### Installation
```bash
git clone https://github.com/tobe01/portfolio.git
cd portfolio
npm install
npm run dev
Build
npm run build
npm run preview
Configuration

Deployment

Deployed on Vercel.

Ensure SPA routing is configured if using React Router (e.g., vercel.json rewrite to /index.html).
